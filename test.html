Прецизійний регулятор серії SA-RP здатний забезпечити високу точність налаштування тиску повітря: 0,2% від діапазону регулювання. Дані регулятори використовуються для системи дозування, точного регулювання зусилля, регулювання інтенсивності подачі краски для пістолетів порошкового фарбування.

Комплект поставки:

  1. Прецизійний регулятор мембранного типу
  2. Механічний манометр заданого діапазону
  3. Кріплення для монтажу вертикальної поверхні

Переваги пневматичних редукторів SA-RP:

  • Висока повторюваність налаштування тиску +/-0,5% від діапазону
  • Висока лінійність підтримання тиску +/-1% від діапазону
  • Висока точність регулювання тиску +/-0,5% діапазону

Додаткова інформація

Конструктивно SA-RP є двомембранним регулятором із зворотним зв'язком по вихідному тиску, тому на відміну від класичних моделей пневматичних регуляторів (серії RN, RU, RM) прецизійні регулятори здатні утримувати задану величину вихідного тиску незалежно від витрати на споживачі.

Під час роботи прецизійного регулятора SA-RP відбувається постійне скидання стисненого повітря через спеціальні отвори. Об'єм скидуваного повітря не більше 5 нл/хв. Це необхідно для нормальної роботи регулятора, тому не закривайте отвори регулятора.

Прецизійний регулятор вимогливий до якості повітря. Розмір твердих частинок у повітрі, що подається в регулятор, не повинен перевищувати 5 мкм. Повітря не повинно містити залишків компресорного мастила. Маслорозпилювач при необхідності має бути встановлений після прецизійного регулятора, а не перед.

Ми рекомендуємо використовувати наступну збірку для забезпечення довготривалої безперебійної роботи даного приладу.

Для швидкого та зручного відключення та підключення повітря до регулятора використовується клапан SV-I, який є у супутніх товарах.
